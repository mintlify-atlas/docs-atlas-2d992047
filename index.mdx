HelloGitHub is a monthly curated newsletter that shares interesting, entry-level open-source projects from GitHub. Published on the **28th of every month**, each issue features projects organized by programming language — from Python and Go to C++, Rust, JavaScript, and beyond. Whether you're a beginner discovering open source for the first time or an experienced developer hunting for inspiration, HelloGitHub helps you find the joy of open source.
