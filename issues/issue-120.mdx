HelloGitHub Issue \#120 is the March 2025 edition of the monthly newsletter. This issue spans C, C#, C++, Go, and Java, and features projects as varied as a tic-tac-toe game squeezed into a single `printf` call, the game engine behind Stardew Valley, a Google DeepMind physics simulator, and an open-source model-rocket flight simulator. Read the full issue on GitHub or the official website, or explore the highlights below.

## Highlights

<CardGroup cols={2}>
  <Card
    title="printf-tac-toe — Tic-Tac-Toe in One printf"
    icon="terminal"
    href="https://github.com/carlini/printf-tac-toe"
  >
    A creative showcase of C language tricks that implements a complete tic-tac-toe game — including game logic, variable assignment, win detection, and board rendering — inside a single `printf` call. Supports number-key moves and board refresh.
  </Card>
  <Card
    title="MonoGame — Cross-Platform .NET Game Framework"
    icon="gamepad"
    href="https://github.com/MonoGame/MonoGame"
  >
    The open-source implementation of Microsoft's XNA framework, powering games like Stardew Valley, Celeste, and Streets of Rage 4. Write C# once and deploy to Windows, macOS, Linux, iOS, Android, PlayStation, Xbox, and Nintendo Switch.
  </Card>
  <Card
    title="mujoco — High-Performance Physics Engine"
    icon="robot"
    href="https://github.com/google-deepmind/mujoco"
  >
    Google DeepMind's general-purpose physics simulation engine, specializing in complex articulated structures. Features an interactive visualizer, MJCF scene description, and Python/Rust/C++ bindings. Used in robotics, embodied AI, biomechanics, and ML research.
  </Card>
  <Card
    title="DriverStoreExplorer — Windows Driver Manager"
    icon="hard-drive"
    href="https://github.com/lostindark/DriverStoreExplorer"
  >
    A portable, open-source Windows driver management tool for viewing, installing, uninstalling, and backing up drivers. Automatically compares driver versions to identify outdated or unused drivers, making driver cleanup straightforward.
  </Card>
  <Card
    title="openrocket — Model Rocket Simulator"
    icon="rocket"
    href="https://github.com/openrocket/openrocket"
  >
    A free, open-source model rocket simulation tool with an intuitive design interface. Supports six-degrees-of-freedom flight simulation and displays altitude, velocity, and acceleration in real time during the simulation. Runs on Windows, Linux, and macOS.
  </Card>
  <Card
    title="STranslate — Desktop Translation & OCR"
    icon="language"
    href="https://github.com/STranslate/STranslate"
  >
    A WPF-based Windows translation and OCR tool integrating Google, DeepL, OpenAI, and other translation services. Supports selected-text translation, screenshot recognition, full-document translation, vocabulary management, and a plugin marketplace.
  </Card>
</CardGroup>
