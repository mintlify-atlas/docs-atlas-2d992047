The latest HelloGitHub issue is published on the 28th of each month. Each edition curates 20–40 beginner-friendly, interesting open-source projects across a broad range of programming languages — from systems tools and game engines to productivity apps and AI utilities. Below is a guide to accessing the most recent issue and navigating its content.

## Current Issue: \#123

HelloGitHub Issue \#123 is the June 2025 edition. This issue spans C, C#, C++, Go, JavaScript, Kotlin, Python, Rust, Swift, and AI categories. Standout picks include **keyd** — a Linux system-level keyboard remapper that works across X11, Wayland, and TTY; **mbedtls** — a compact, portable TLS library designed for IoT and embedded devices; **PaperTodo** — a borderless, minimal Windows sticky-notes app with auto-save; **luanti** — an open-source voxel game engine for building 3D block games with Lua; **gopass** — a GPG-encrypted, Git-synced command-line password manager designed for teams; **react-scan** — a zero-config React performance debugger that visually highlights unnecessary re-renders; **black** — the official PSF Python code formatter that eliminates style debates; and **Scrapling** — a self-adapting Python web-scraping framework that survives page redesigns automatically.
